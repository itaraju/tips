dealing with white-spaces in names, using find/xargs:
find -iname 'pattern' |xargs -I{} rm {}

similar:
 find ~/gs-v1/ -iname '._*' |xargs -I{} rm -v {}

Replacing underlines with spaces on all files within a directory
    for f in `ls -1 *_*`; do mv "$f" "$(echo "$f" | sed "s/_/ /g")"; done

retornar colunas separadas por ' '
awk -F ' ' '{print $1}' 

tentar:
perl -F" " -nae 'print "$F[1] \n";' < t

retirar aspas simples de começo ou fim de linhas:
perl -ne 'chomp; $_=substr($_,1) if /^'\''/; chop if /'\''$/; print "$_\n";'

obter colunas por posição:
cut

-- PDF
extração de páginas:
$ pdftops afile.pdf - | psselect -p11-14 | ps2pdf - file-p11-14.pdf
ou
$ pdftk A=afile.pdf cat A11-14 output file-p11-14.pdf

merging:
psmerge

$ pdftops file1.pdf 
$ pdftops file2.pdf
$ psmerge file1.ps file2.ps | ps2pdf - finalfile.pdf
$ rm file1.ps file2.ps

para merge de vários arquivos:
$ for f in `ls *.pdf`; do pdf2ps "$f"; done
$ echo *.ps|xargs psmerge -ofinalfile.ps #não funcionou este

Alternativa:
pdfjam
